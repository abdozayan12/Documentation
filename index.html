JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in others...

In contrast to Java's compile-time system of classes built by declarations...

JavaScript is a very free-form language compared to Java...

For example, JavaScript uses console.log() for logging to the console.

Another example is the function keyword used to define a function.

Hello world

To get started with writing JavaScript, open the Scratchpad and write your first "Hello world" JavaScript code:

function greetMe(yourName) { alert("Hello " + yourName); }

greetMe("World");

Example of calling the function:

greetMe("Hello");

Variables

You use variables as symbolic names for values in your application.

Example of declaring a variable:

let x = 10;

Another example:

const name = "JavaScript";

Global variables

Global variables are properties of the global object. For example:

window.myVar = "Global variable";

Data types

The latest ECMAScript standard defines seven data types:

  • Boolean: true and false.
  • null: A special keyword denoting a null value.
  • undefined: A top-level property whose value is undefined.
  • Number: Example: 42 or 3.14159.
  • String: Example: "Howdy"
